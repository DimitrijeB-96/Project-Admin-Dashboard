My Odin Projects

First project: Recipes

This was introduction project, and before this project focus was at setting up Ubuntu Linux operating system on VM and some basics on how to use it. After basics with operating system next focus was at setting up code editor, Git and GitHub. And finally introduction at HTML and CSS. At this project majority of code is in HTML and just few lines in CSS.

Second project: Landing Page

After first project "Recipes", focus was at placement of content at the page. Differences between Block and Inline box model. Implementation of display Flexbox with it's alignment properties like growing, shrinking, align and justify.

Third project: Rock Paper Scissors

This was first project in section JavaScript Basics and this project was done in two parts. In first part game of Rock Paper Scissors was done in console, while input was at pop-up window. Second part was revisited after curriculum for DOM manipulation and focus was at getting game played from console to the page.

Fourth project: Etch-a-Sketch

This project was done after revisiting Rock Paper Scissors to implement DOM. Focus of this project was at implementing everything that I've learned of JavaScript so far.

Fifth project: Calculator

This was last project from Foundations part of curriculum. The only new thing for this project was introduction at objects in JavaScript. This was the hardest project in Foundations if you ask me, but it was wort it!

The Sixth project: Sign-up Form

First project in Full Stack JavaScript path. For this project focus was getting more advanced HTML and CSS. All positioning was done with Flexbox, but I've added dynamic scalling to test my skills since I haven't used it before for the previous projects.
